Las clases usadas fueron:

- Practica08: contiene el método main y los registros guardados y el menú interactivo.

- Tienda: es la clase gestora, con las Collections y los métodos relacionados a estas.

- Producto: es una clase para manejar los objetos que se usan

- Cliente: es una clase para manejar los objetos que se usan, sobre todo para manejar una lista de productos ligada a ellos

- CompararMarca: es una clase que implementa Comparator, se usa para comparar las marcas de 2 productos

- CompararCategoria: es una clase que implementa Comparator, se usa para comparar las categorías de 2 productos

Se implemento un filtrado compuesto de una marca y menor a x precio, util por si consideras que una marca es de buena calidad pero con precios altos
