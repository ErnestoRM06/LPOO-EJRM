Las clases usadas fueron:

- Practica09: contiene el método main y el menú como una linea de comandos.

- Archivos: es la principal clase gestora, ayuda a manejar los archivos de texto, binarios, backups y directorios.

- Paciente: es la clase a serializar.

- ProcesadorCSV: es una clase que maneja los archivos CSV, ya sea crearlos o leerlos.

- ProcesadorXML: es una clase que maneja los archivos XML, ya sea crearlos o leerlos.

- Serializador: es una clase que maneja los objetos serializados, ya sea el crearlos o el leerlos.

Usar otros formatos aparte de CSV puede ser útil para dar otros formatos a la información o para que sea compatible con otras funciones, como lo son JSON y XML, que se usan como bases de datos no regularizadas.
