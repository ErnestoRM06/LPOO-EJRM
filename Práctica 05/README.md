Primero, las clases usadas fueron:

- Practica05: contiene el método main y las pruebas hechas.

- Planta: es la clase abstracta, tiene los atributos de nombre y altura, así como métodos relacionados y el método de plantar.

- Macroscopico: es la interfase, tiene el método de crecer

- Flor: es una clase que hereda de ambas, que además tiene el atributo de color

- Árbol: es una clase que hereda de ambas, tiene de atributo si tiene semillas o fruto

- Tuberculo: es una clase que hereda de ambas, tiene un atributo de peso

La clase Planta tiene un método para llamar a los métodos abstractos llamado información, el cual escribe la información básica de la planta, lo cual ayuda a no hacerlo en cada instancia o clase.
