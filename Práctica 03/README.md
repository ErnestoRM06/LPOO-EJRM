Primero, las clases usadas fueron:

- Practica03: contiene el método main y las pruebas hechas.

- Alumno: es la clase con atributos private.
En esta clase implemente las siguientes reglas:
- Para la calificación, que no sea menor a 0 ni mayor a 100, por que así es.
- Para la matricula, que no sea menor o igual a 0, ya que así se suele usar.
- Para el nombre, solo aceptamos letras mayúsculas y espacios, ya que esto ayuda a no tener que diferenciar las letras mayúsculas y minúsculas.

- Profesor: es la clase con atributos protected.
En esta clase implemente las siguiente reglas:
- Para el nombre, lo mismo que con Alumno.
- Para el teléfono, que solo contenga 10 números, sin espacios, guiones, y sin el prefijo regional, para que sea más fácil de manejar.

- auxiliar: es una clase en otro paquete para comprobar los atributos protected.
